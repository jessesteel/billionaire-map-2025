# 🌍 Global Billionaire Intelligence Dashboard (2025)

An advanced, interactive 2D map visualization of the global billionaire network and their interconnected corporate empires. 

Built with **React 18**, **Leaflet.js**, and a high-performance **Canvas/SVG hybrid rendering engine**.

## 🚀 Key Features

- **Institutional Titans**: Visualizing the "Big Three" (BlackRock, Vanguard, State Street) and their massive global network of majority-shareholder connections.
- **Dynamic Portfolio Enrichment**: Automated SPARQL extraction pipeline that crawls Wikidata to uncover hidden corporate subsidiaries.
- **Tactical Anti-Collision Math**: 
  - **Fermat's Spiral**: Packing billionaires in dense urban hubs (NYC, London, SF) without visual overlap.
  - **Structured Grid-Rank**: Organizing billionaire clusters by wealth rank for a cleaner, searchable interface.
- **Cinematic Network Traversal**: Click any node to instantly isolate their entire network with "marching ants" capital flow animations.
- **Real-Time Global Search**: Debounced, high-performance filtering across 3,000+ entities with an instant "Dark-Out" effect for non-matches.

## 🛠 Tech Stack

- **Frontend**: React 18 (CDN), Leaflet.js
- **Rendering**: Optimized HTML5 Canvas for map points, SVG for active network paths.
- **Data Engineering**: Python/SPARQL extraction and LLM-assisted knowledge injection.
- **Styling**: Glassmorphic CSS with vanilla variables and high-fidelity dark-mode aesthetics.

## 📦 Getting Started

This is a static web application. To run it locally:

1. Clone the repository: 
   ```bash
   git clone https://github.com/YOUR_GITHUB_USERNAME/billionaire-map-2025.git
   ```
2. Simply open `index.html` in your browser (preferably via a local static server like `npx serve .` or Live Server to avoid CORS issues with the JSON fetch calls).

## 📊 Data Disclosure

The primary billionaire dataset is derived from the **Forbes 2025 Real-Time List**, with deep-web corporate expansions conducted via automated Wikidata SPARQL queries and manual portfolio mappings for private equity and institutional titans.
